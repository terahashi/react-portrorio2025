🔶SPAメモ(ReaxtはSPAです)
・Reactは「SPA(シングルページアプリケーション)」であり、SPAはHTMLなどと違って【再レンダリングしてDOMを切り替える。
なのでブラウザはページ遷移をしないでコンテンツを動的に更新する。】
1:初回アクセス時に単一のHTML、CSS、JavaScriptを読み込む。﻿
2:以降は、ユーザーの操作に応じてJavaScriptがAPI経由でデータ（JSONなど）だけを取得。﻿
3:画面の変更が必要な部分だけを更新（:差分更新）。﻿

✅iPhone 実機で Safari デバッグする方法（Mac 必須）

①準備：Mac 側の設定
・Safari の「開発メニュー」を ON にする
1:Mac の Safari を起動
2:メニューバー → Safari → 設定
3:「詳細」タブへ
4:1番下の“メニューバーに“開発”メニューを表示”にチェックを入れる
これで Safari のメニューバーに「開発」が追加される。



②iPhone 側の設定
Web 検査（Web Inspector）を ON にする
1:iPhone の 設定アプリを開く
2:「Safari」へ
3:一番下 → 「詳細」
4:Webインスペクタ（Web Inspector）を ON



③iPhone と Mac を USB で接続
Lightning ケーブルiPhone を Mac へ接続。
・初めての場合：
1:iPhone に 「信頼しますか？」→ “信頼”
2:Mac の Finder に iPhone が出たら「このコンピュータを信頼」のような確認が出るので承認


④実機デバッグ開始（ここが本番）
iPhone の Safari であなたのサイトを開く（ローカル開発なら http://localhost:5173とか）
・次に Mac の Safari で：
1:メニューバーの **「開発」**をクリック
2:自分の iPhone のデバイス名が表示される
3:そこに 開いている URL の一覧が出る

例：
iPhone（ka-ta の iPhone）
　┣ localhost:5173
　┗ google.com

4:調べたい URL をクリックすると…iPhone 実機のサイトが、Mac Safari の DevTools（Chromeみたいな開発者ツール）に出現！
